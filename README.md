# MarketDataServices
SOA Course Project

- archived here for reference
- protocol defined in WSDL
- handles error in a detialed way

main functions:

1. importMarketData() - get data from a link (when testing, we used files provided by lecturer on a uni server)
2. downloadFile() - presents entity in XML/HTML/JSOM format to the requester
3. convertMarketData() - currency conversion
4. yahooExchangeRate() - get exchange rate from yahoo
